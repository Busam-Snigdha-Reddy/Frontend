## Key Definitions

### HTML
HTML (HyperText Markup Language) is the standard markup language used to create web pages. It defines the structure of content using elements like headings, paragraphs, links, images, and layout containers.

### CSS
CSS (Cascading Style Sheets) is used to style HTML content. It controls layout, colors, spacing, fonts, and responsiveness.

### Grid
CSS Grid is a layout system that allows web developers to create two-dimensional grid-based layouts. It provides rows and columns, and items can be positioned precisely inside the grid using properties like `grid-template-columns`, `grid-template-rows`, `grid-gap`, and `grid-area`.

### Bootstrap
Bootstrap is a popular front-end framework for building responsive websites quickly. It includes prebuilt CSS classes, components, and utilities for layout, typography, forms, buttons, navigation, and responsive grids.

### Responsive Design
Responsive design means creating web pages that adapt to different screen sizes and devices. Common techniques include flexible layouts, media queries, and responsive frameworks like Bootstrap.
